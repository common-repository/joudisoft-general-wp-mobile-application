WordPress General Mobile Application Plugin helps WordPress webmasters to build their mobile application in minutes

== Description ==

WordPress General Mobile Application Plugin makes it easy to create native Android mobile application for any WordPress website, well formed application, easily publish it to Google Play Store and other application stores, From simple websites to complex WordPress websites, Our plugin makes it easy to build your own mobile application in minutes, joudiSoft infrastructure will take care of building your mobile app for you.

== Before using the Plugin ==

By creating application you create application into wordpress end-servers and in order to build your application based on joudiSoft Application Box servers, the plugin will send the following information about your WordPress blog to joudiSoft' servers:

- WordPress Blog Name
- WordPress Blog URL
- The details you fill into The plugin screen

You must read and agree to the following:

- Terms of Services - https://joudisoft.com/legal/terms/
- Privacy Policy - https://joudisoft.com/legal/privacy-policy/


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/appbox-wordpress-general` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Start up using the plugin through menu item `joudiSoft General WordPress Mobile Application`
4. You may need to visit the following URL to learn more about using the plugin: https://docs.joudisoft.com/application-box/wordpress-general/

== Frequently Asked Questions ==

1- How to build My Application
Just follow up the simple steps in `joudiSoft General WordPress Mobile Application`, After building process you will see the download link into the plugin screen left section you can easily download your APK and test out and publish on google play store.

2- How will My Application work
Your application will be linked to your website / blog so whenever you post a new feed or change colors / info .. etc users will see your changes and updates directly at your application without the need of re-building your mobile application again.

3- Can i change My Mobile Application Logo
Yes, you can use your own logo, just go to `Branding` section.

4- Can i change theme colors
Yes, We give the option to customize theme colors, you can change it to match your blog or company website.

5- Can i manage and create Application Menu
Yes, you can manage the native menu and /or/ create your own customized Menu items from `Native Menu` and `Custom Menu` sections.

6- Can people contact me from my Application
Yes, we have created a contact us section in application you can turn it on and put your contact info in `Contact` section.

7- How to put my social networks links into application
Using `Social` section you can easily put your social networks links and turn it on your application.

8- Google play upload fail with following error
You uploaded an APK that is signed with a key that is also used to sign APKs that are delivered to users. Because you are enrolled in App Signing, you should sign your APK with a new key before you upload it.
Don’t enable App Signing – <a href="https://support.google.com/googleplay/android-developer/answer/7384423?hl=en">follow this article</a>
